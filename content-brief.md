# David Smith Portfolio Site — Complete Content Brief

---

## Build Instructions — Approach A

Build the complete site using placeholder images in all image slots. Use a consistent placeholder style — a dark grey rectangle with the image filename as a label centered in it (e.g., `booth-panels.png`). This allows the layout, hierarchy, and spacing to be evaluated before final images are dropped in.

All placeholders should be sized proportionally to the actual content they'll hold:

| Content Type | Aspect Ratio |
|---|---|
| Full-width hero pieces | 16:9 or wider |
| Portrait brochures | ~3:4 |
| Square social cards | 1:1 |
| Two-panel booth | ~3:1 (wide panoramic) |

## Case Study Pages — Global Structure

Each case study page follows this structure top to bottom:

1. Back navigation link ("← Selected Work")
2. Project title (large)
3. Tag line (role · employer · year)
4. Introduction paragraph
5. Deliverables in the specified order (hero pieces first, supporting pieces after)
6. Each deliverable: image/placeholder → caption below
7. Bottom navigation: "← Previous" / "Next →" between case studies

**Image hover behavior:** Subtle scale up (1.02) with smooth transition on desktop. No overlay text. Clean and minimal.

**Mobile behavior:** All two-column grids collapse to single column. Hero images remain full width. Navigation collapses to hamburger menu.
